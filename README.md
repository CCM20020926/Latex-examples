# 常用Latex 排版示例

**图排版：** 参见 [figures/](figures/)

**表排版：** 参见 [tables](tables/)

**引用图表：**

\ref{tab_or_fig_name}  只插入编号不插入标签类型（table/figure）

\autoref{tab_or_fig_name} 插入标签类型 + 编号（如 Table 1, Figure 1）

**添加上角标：** $^{1}$

**作者单位显示示例：**

author1$^{1}$, author2$^{1}$, author3$^{2}$ \\
$^{1}$XXX,Beijing,China \\
$^{2}$YYY,Beijing,China \\
{author1,author2}@xxx.cn,{author3}@yyy.cn
